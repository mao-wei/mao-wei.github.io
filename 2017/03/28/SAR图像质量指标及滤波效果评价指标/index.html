SAR图像质量指标及滤波效果评价指标

2017-03-28 毛威

摘要：此篇主要介绍评价SAR图像性能的指标、以及评价滤波算法对图像滤波效果的指标，主要内容列为以下几点：

  • SAR图像性能指标

    • 空间性能指标
    • 辐射性能指标
    • 试验中用到的图像性能指标

  • 滤波函数效果评价指标

    • 平滑指数
    • 边缘保持指数
    • 等效视数

1. 空间性能指标

SAR图像的空间性能指标描述了SAR图像的空间性能，主要包括：

  • 观测带宽
  • 入射角
  • 空间分辨率
  • 定位精度
  • 几何畸变

2. 辐射性能指标

SAR图像的辐射指标描述SAR图像的辐射性能，主要包括：

  • 动态范围
  • 等效噪声
  • 等效视数
  • 辐射测量精度
  • 信噪比
  • 模糊度
  • 峰值旁瓣比
  • 积分旁瓣比

3. 试验中将会用到的图像性能指标

在对SAR影像进行预处理后，需要对SAR影像预处理后的结果图像进行评价，而之前所列举的指标大部分都是一幅图像所固有的，对于图像预处理前后效果的对比起不到什么作用，这里主要选取那些能够体现图像预处理前后质量的指标来作为试验中的评价标准，包括：灰度均值、方差、等效视数

  • 灰度均值：灰度均值是指整个图像的平均灰度，反映了图像的实际测量值中所包含的目标平均后向散射特性。其计算方法为对图像中所有像素的值求平均。

  • 方差：图像方差反映图像的不均匀性，其计算方法为对图像中所有像素灰度值与灰度均值之差的平方求平均

  • 等效视数：等效视数度量了图像区分具有不同后向散射特性区域的能力，是衡量一幅SAR图像斑点噪声相对强度的一种指标，等效视数越大，表明图像上斑点越弱。其计算方法为对SAR图像的灰度均值与方差求商再平方。

以上三个指标将会在试验中作为评价图像质量的评价指标，为后续的SAR图像特征提取提供基础支撑。

4. 滤波函数效果评价指标

SAR图像噪声与普通的光学影像噪声在产生原因以及噪声分布模型上都存在较大的区别，因此对于SAR图像的滤波效果评估也和普通图像有所区别。考虑到滤波的过程中很容易使SAR图像的边缘产生模糊，SAR图像的强度信息丢失，我们采用平滑指数、边缘保持数、以及等效视数来对滤波效果进行评价。

  • 平滑指数：平滑指数反映了滤波算法对图像噪声的平滑作用，取值范围为0~1，其值为滤波后图像的均值与滤波后图像的标准差的比值。（公式见《SAR图像处理与目标识别》P28）

  • 边缘保持数： 可以简单理解为滤波后的图像的边缘指数与滤波前的图像的边缘指数的比值。单个像素点的边缘指数的计算方法为该像素点与其边缘两像素（成直角）的灰度值之差的平方和再开方，一幅图像的边缘指数可以类推为所有像素的边缘指数之和（公式见《SAR图像处理与目标识别》P28）

  • 等效视数： 前文已经提到，在这里值得注意的是，等效视数有一个比列系数，这里我们用到的比例系数为0.5227，计算的是滤波后图像的均值与滤波后标准差的比值乘上比例系数后所得结果再平方。（注：此处对于0.5227存在疑问）

这三个指标能对滤波函数的效果进行很好的评价，是滤波试验中用到的对于滤波效果的主要评价指标。
